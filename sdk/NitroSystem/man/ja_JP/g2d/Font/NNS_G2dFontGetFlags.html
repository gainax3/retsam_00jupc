NNS_G2dFontGetFlags

構文


#include <nnsys/g2d/g2d_Font.h>

u8 NNS_G2dFontGetFlags(
	const NNSG2dFont* pFont
);

引数

pFont [IN] フォント へのポインタ

返り値

フォントの特徴フラグを返します。

説明

pFont の特徴フラグを取得します。

特徴フラグのフォーマットは以下のようになっています。 

|---- 8 bit ----|
MSB             LSB
+---------+---+-+
|         |   | |
+---------+---+-+

|- 5 bit -|---|-|
     |      |  |
  reserve   |  1 bit 縦書き用フォントか否か
            2 bit グリフの回転
               0: 時計回り   0 度
               1: 時計回り  90 度
               2: 時計回り 180 度
               3: 時計回り 270 度
