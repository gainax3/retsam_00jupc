NNS_G2dGetUserExAnimFrameAttr

構文

#include <nnsys/g2d/fmt/g2d_Anim_data.h>

static inline const NNSG2dUserExAnimFrameAttr* 
NNS_G2dGetUserExAnimFrameAttr
( 
    const NNSG2dUserExAnimSequenceAttr* pAnimSeqAttr,
    u16                                 idx 
);

引数

pAnimSeqAttr [IN] アニメーションシーケンスユーザ拡張アトリビュート
idx [IN] アニメーションフレームの番号

返り値

アニメーションフレームユーザ拡張アトリビュート

説明

アニメーションシーケンスユーザ拡張アトリビュート から アニメーションフレームユーザ拡張アトリビュート を取得します。
不正な番号が指定された場合は、NULLを返します。
ユーザ拡張アトリビュートを含むデータを出力するためには、-oua オプションを指定してデータを出力する必要があります。
