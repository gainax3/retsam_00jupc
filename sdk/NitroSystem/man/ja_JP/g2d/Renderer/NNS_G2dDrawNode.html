NNS_G2dDrawNode

構文

#include <nnsys/g2d/g2d_Renderer.h>
void NNS_G2dDrawNode( const NNSG2dNode* pNode );

引数

pNode [IN] ノード

返り値

なし

説明

ノード（に関連付けられた描画物）を描画します。

Begin End Rendering 内 で 呼び出して下さい
現在は NNS_G2D_NODETYPE_CELL タイプの Nodeのみをサポートしています。

参照

NNS_G2dDrawMultiCellAnimation
 NNS_G2dDrawCellAnimation

履歴

2004/05/28 初版
