NNS_G2dCharCanvasClearArea

構文


#include <nnsys/g2d/g2d_CharCanvas.h>

void NNS_G2dCharCanvasClearArea(
	const NNSG2dCharCanvas* pCC,
	int cl,
	int x,
	int y,
	int w,
	int h
);

引数

pCC [IN] CharCanvas へのポインタ
cl [IN] 塗りつぶしに用いるカラーNo.
x [IN] 塗りつぶし矩形の左上隅座標 (CharCanvas座標系)
y [IN] 塗りつぶし矩形の左上隅座標 (CharCanvas座標系)
w [IN] 塗りつぶし矩形の幅 (ピクセル単位)
h [IN] 塗りつぶし矩形の高さ (ピクセル単位)

返り値

なし。

説明

CharCanvas 上の矩形を指定された色で塗りつぶします。
座標は CharCanvas 左上隅を(0,0)とする座標系をとります。

この関数は NNS_G2dCharCanvasClear() に比べて非常に低速なので CharCanvas 全体を塗りつぶす場合には NNS_G2dCharCanvasClear() を使用してください。
