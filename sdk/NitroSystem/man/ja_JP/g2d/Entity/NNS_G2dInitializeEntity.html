NNS_G2dInitializeEntity

構文

#include <nnsys/g2d/g2d_Entity.h>
void NNS_G2dInitializeEntity( 
                        NNSG2dEntity*               pEntity, 
                        void*                       pDrawStuff, 
                        const NNSG2dEntityData*     pEntityData, 
                        const NNSG2dAnimBankData*   pAnimDataBank );

引数

pEntity [OUT] エンティティ
pDrawStuff [IN] 描画用データ( NNSG2dCellAnimation or NNSG2dMultiCellAnimation )
pEntityData [IN] エンティティ定義データ
pAnimDataBank [IN] アニメーションデータバンク

返り値

なし

説明

エンティティを初期化します。
pDrawStuff や pAnimDataBank は Entity のタイプによって適切なものを渡す必要があります。 条件に合わない入力があった場合はアサートに失敗します。
本関数はNNS_G2dInitEntity()へと改名されました。本関数は、互換性維持のため別名として残されます。

参照

NNS_G2dInitEntity
 NNS_G2dSetEntityCurrentAnimation
