NNS_G2dInitAnimControllerCallBackFunctor

構文

#include <nnsys/g2d/g2d_Animation.h>
void NNS_G2dInitAnimControllerCallBackFunctor( 
        NNSG2dAnimController*     pAnimCtrl );

引数

pAnimCtrl [OUT] NNSG2dAnimController 実体

返り値

なし。

説明

コントローラのコールバックを初期化します。
NNS_G2dInitAnimCallBackFunctor( ) を内部で呼び出します
本関数はNNS_G2dInitAnimCtrlCallBackFunctor()に改名されました。互換性維持のため、過去の関数は別名として残されています。
