NNS_G2dSetCallBackFunctor

構文

#include <nnsys/g2d/g2d_Animation.h>
void NNS_G2dSetCallBackFunctor( 
                      NNSG2dAnimController*   pAnimCtrl, 
                      NNSG2dAnmCallbackType   type, 
                      u32                     param, 
                      NNSG2dAnmCallBackPtr    pFunc );

引数

pAnimCtrl [OUT] NNSG2dAnimController 実体
type [IN] コールバック呼び出しの種類
param [IN] コールバック関数の引数となる u32 データ
pFunc [IN] コールバック関数ポインタ

返り値

なし。

説明

アニメーションコントローラにコールバックを設定します。
コールバックはコントローラ実体毎に1種類のみ登録可能です。

本関数の引数 type に NNS_G2D_ANMCALLBACKTYPE_SPEC_FRM を指定するのは不正です。
その場合は NNS_G2dSetAnimCtrlCallBackFunctorAtAnimFrame() を 使用する必要があります。

本関数は NNS_G2dSetAnimCtrlCallBackFunctor() に 改名されました。本関数は、過去のバージョンとの互換性維持のため、別名としてのこされます。

参照

NNS_G2dSetAnimCtrlCallBackFunctor
 NNS_G2dSetAnimCtrlCallBackFunctorAtAnimFrame
