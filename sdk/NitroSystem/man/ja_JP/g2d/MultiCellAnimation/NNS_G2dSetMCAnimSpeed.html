NNS_G2dSetMCAnimSpeed

構文

#include <nnsys/g2d/g2d_MultiCellAnimation.h>
void NNS_G2dSetMCAnimSpeed
                           (
                               NNSG2dMultiCellAnimation*   pMultiCellAnim, 
                               fx32                        speed  
                            );

引数

pMultiCellAnim [OUT] マルチセルアニメーション
speed [IN] アニメーションスピード

返り値

なし

説明

マルチセルアニメーションをのアニメーションスピードを設定します。
内部で、すべての構成セルアニメーションに対してNNS_G2dSetCellAnimationSpeed()をコールしています。

参照

NNS_G2dSetCellAnimationSpeed

履歴
