NNS_G2dGetMCNumNodesRequired

構文

#include <nnsys/g2d/g2d_MultiCellAnimation.h>
u16 NNS_G2dGetMCNumNodesRequired( 
                                const NNSG2dMultiCellAnimSequence*    pMultiCellSeq, 
                                const NNSG2dMultiCellDataBank*        pMultiCellDataBank );

引数

pMultiCellSeq [IN] マルチセルアニメーションシーケンス
pMultiCellDataBank [IN] マルチセルデータバンク

返り値

マルチセルアニメーション中のマルチセルの持つ最大ノード数

説明

マルチセルアニメーションを構成するマルチセルを走査し アニメ中のマルチセルの持つ最大ノード数を返します
似たような用途に使用する、NNS_G2dGetMCBankNumNodesRequired()も用意されています。 こちらは、マルチセルデータバンク中のマルチセルの持つ最大ノード数を返します。
用途に合わせてご利用ください。
