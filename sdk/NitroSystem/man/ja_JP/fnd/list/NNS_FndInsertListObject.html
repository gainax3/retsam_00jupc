NNS_FndInsertListObject

構文

#include <nnsys/fnd.h>
void NNS_FndInsertListObject( NNSFndList* list, void* target, void* object );

引数

list リスト構造体へのポインタ
target 挿入したい位置に繋がれているオブジェクトへのポインタ
object リストに繋げたいオブジェクトへのポインタ

返り値

なし。

説明

objectで指定されたオブジェクトをtargetで指定されたオブジェクトの前に挿入します。 targetにNULLを指定した場合には、オブジェクトはリストの最後に追加されます。

参照

NNS_FndAppendListObject, NNS_FndPrependListObject, NNS_FndRemoveListObject
