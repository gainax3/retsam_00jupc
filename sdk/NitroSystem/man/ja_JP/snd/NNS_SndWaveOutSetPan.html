NNS_SndWaveOutSetPan

構文

#include <nnsys/snd.h>
void NNS_SndWaveOutSetPan( NNSSndWaveOutHandle handle, int pan );

引数

handle 波形再生ハンドルです。
pan パン（定位）です。値の範囲は、0(左)〜 64(中央)〜 127(右) です。

返り値

なし。

説明

波形再生のパン（定位）を変更します。 すでに停止中の波形再生ハンドルhandleに対し、この関数を呼びだしても何もしません。

この関数は、再生中にパンを変更するために呼びだします。 再生開始時のパンを設定したい場合は、 NNS_SndWaveOutStart関数 の引数で設定してください。
