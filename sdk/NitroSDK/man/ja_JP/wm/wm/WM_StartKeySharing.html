WM_StartKeySharing

構文

#include <nitro/wm.h>

WMErrCode WM_StartKeySharing( WMKeySetBuf* buf, u16 port );

引数

buf キー情報を格納するバッファへのポインタを指定します。
実体は WMDataSharingInfo 構造体へのポインタです。
port 使用する port 番号を設定します。12から15までのうち、他で使用していない port 番号を選択してください。

返り値

WMErrCode列挙型の処理結果を返します。

説明

キーシェアリング機能を有効にします。WMがMP_PARENTステート、もしくはMP_CHILDステートの場合に実行可能です。 この関数は非同期関数から同期関数へ変更されました。
WM_StartKeySharing, WM_EndKeySharing, WM_GetKeySet は 過去との互換性のために残されており、今後廃止される予定です。Data Sharing の各関数を代わりに使用してください。

参照

WM_EndKeySharing, WM_GetKeySet
