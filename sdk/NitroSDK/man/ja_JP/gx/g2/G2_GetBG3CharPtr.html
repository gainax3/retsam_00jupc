G2_GetBG3CharPtr

構文

#include <nitro/gx/g2.h>

void *G2_GetBG3CharPtr();

引数

なし

返り値

BG3のキャラクタデータの格納領域へのポインタ

説明

メイン2DエンジンのBG3キャラクタデータの格納領域へのポインタを返します。つまり返り値は、

キャラクタＢＧの場合：

HW_BG_VRAM
+ （GX_SetBGCharOffsetで指定したGXBGCharOffset列挙型の値名の末尾の値）
+ （G2_SetBG3ControlText等で指定したGXBGCharBase列挙型の値名の末尾の値）

になります。ビットマップBGの場合はキャラクタデータは存在しないのでNULLを返します。

参照

G2_SetBG3ControlText, ..., G2_SetBG3Priority, G2_BG3Mosaic, G2_GetBG3ScrPtr, G2_SetBG3Offset
