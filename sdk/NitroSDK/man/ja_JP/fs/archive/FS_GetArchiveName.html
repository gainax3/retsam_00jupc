FS_GetArchiveName

構文

#include <nitro/fs.h>

static inline const char* FS_GetArchiveName(const FSArchive *p_arc);

引数

p_arc FSArchive構造体のアドレス

返り値

アーカイブに付けられた名前を表す文字列へのポインタを返します。
名前を登録していないアーカイブは、空文字列 "" を返します。

説明

アーカイブを登録した時につけた名前を文字列で取得します。
得られた文字列ポインタは一時的なもので、次にこのアーカイブに対して
FS_RegisterArchiveName関数またはFS_ReleaseArchiveName関数を呼び出したとき無効となります。
また、文字列の中身を変更することはできません。

参照

FSArchive
FS_RegisterArchiveName FS_ReleaseArchiveName
