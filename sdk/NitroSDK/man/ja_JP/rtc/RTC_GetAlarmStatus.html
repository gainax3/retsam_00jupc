RTC_GetAlarmStatus

構文

#include <nitro/rtc.h>
RTCResult RTC_GetAlarmStatus( 
            RTCAlarmChan	chan, 
            RTCAlarmStatus*	status );

引数

chan アラームチャンネルをRTCAlarmChan列挙型で指定します。
status アラーム状態を受け取るRTCAlarmStatus列挙型の変数へのポインタ

返り値

要求が成功した場合に 0 を返します。
要求が失敗した場合には 0 以外を返します。

説明

リアルタイムクロックからアラーム割り込み許可状態を読み出します。
この関数はARM7にアラーム割り込み許可状態読み出し要求を発行し、ARM7からの応答を待って処理結果を返します。
RTC_GetAlarmStatusAsyncの同期版です。

注意

この関数は同期を取る為にPXI割り込みを待ちますので、PXI割り込みを禁止した状態で呼び出さないようにして下さい。
