VEC_Fx16Subtract

構文

#include <nitro/fx/fx_vec.h>

void VEC_Fx16Subtract(
    const VecFx16 * a,
    const VecFx16 * b,
    VecFx16 * ab
);

引数

a ベクトルへのポインタ
b ベクトルへのポインタ
ab ベクトルaとベクトルbの差

返り値

なし

説明

VecFx16型ベクトルの減算を行います。
*a,*b,*abは同じベクトルへのポインタでも構いません。

参照

VecFx16, VEC_Fx16Add
