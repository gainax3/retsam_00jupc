VEC_MultAdd

構文

#include <nitro/fx/fx_vec.h>

void VEC_MultAdd(
    fx32 a,
    const VecFx32* v1,
    const VecFx32* v2,
    VecFx32* pDest
);

引数

a fx32型のスカラー値
v1 ベクトルへのポインタ
v2 ベクトルへのポインタ
pDest 結果が格納されるベクトルへのポインタ

返り値

なし

説明

VecFx32型ベクトルをスカラー倍し、更にベクトルを加算します。結果は、*pDest = a x v1 + v2になります。
*v1,*v2,*pDstは同じベクトルへのポインタでも構いません。

参照

VecFx32, VEC_MultSubtract
