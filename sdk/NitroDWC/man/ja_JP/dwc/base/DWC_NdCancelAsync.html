DWC_NdCancelAsync

構文


#include <nd/dwc_nd.h>

BOOL DWC_NdCancelAsync( void )

説明

現在ダウンロードライブラリが実行中の非同期処理を中断します。

非同期処理の中断完了時にDWC_NdInitAsync関数で指定したDWCNdCallback型のコールバック関数が呼び出されます。このとき、コールバック関数の引数errorとしてDWC_ND_ERROR_CANCELEDが入ります。

引数

なし

返り値

TRUE 成功。非同期処理の中断の完了時にコールバック関数が呼び出されます。
FALSE 失敗、あるいはキャンセルすべき非同期処理がなかった
DWC_GetLastErrorEx関数を使用してエラーの情報を取得してください

参照

DWC_NdInitAsync, DWC_GetLastErrorEx, DWCNdCallback, DWCNdError
